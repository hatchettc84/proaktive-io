# Brand / Product Architecture

## One platform, multiple solution layers
- **ProAktive IO**: platform infrastructure (tenanting, audit, workflows, document intelligence, control libraries, reporting).
- **ProaktivAI**: AI workforce subscription (agent bundles) that run inside IO.
- **InqubatorAI**: a packaged multi-agent workflow experience (researcher → creator → analyst → operator → strategist) built on IO.

## Naming guidance
- IO = the product customers buy for compliance/workflow automation.
- ProaktivAI = agent SKUs/bundles inside IO ("AI Compliance Analyst", "AI Researcher", etc.).
- InqubatorAI = a vertical experience (startup incubator) that demonstrates agent workflows.

## Nonprofit separation
- **ProaktiveIT** should remain clearly separate from for-profit brands.
- Any shared software access should be via documented license or donation (avoid commingling).
