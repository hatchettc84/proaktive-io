# Offer: CMMC L2 / NIST 800-171 Compliance Intelligence Pilot

## Who this is for
Government contractors preparing for **CMMC Level 2** (NIST 800-171).

## Problem
Teams burn weeks creating implementation statements, chasing evidence, and figuring out what’s missing.

## What you get (2-week pilot)
You upload your existing documentation and evidence (policies, SSP sections, procedures, artifacts). We deliver:
- Coverage score by 800-171 family + overall
- Control-by-control status: covered / partial / missing
- Missing evidence checklist (actionable)
- Draft implementation statements for missing/partial controls
- POA&M candidate list
- Exportable report (PDF/DOCX + JSON)

## What we need from you
- Document set (or access) + one technical point of contact
- 60–90 minutes kickoff call

## Success criteria
- Reduce time-to-draft for implementation statements by 70%+
- Clear, prioritized gap list for remediation and evidence collection

## Pricing (initial)
- Pilot onboarding: $1,500–$5,000 (scope-dependent)
- Subscription after pilot: $299–$1,500/mo (tiered)

## Next step
If you share your current SSP/policies, we can return an initial gap report in 48–72 hours.
