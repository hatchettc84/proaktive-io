# ProAktive IO

ProAktive IO is an AI-powered automation platform for governance, risk, compliance (GRC), and operational workflows.

- **Platform layer:** ProAktive IO
- **Workforce layer (agents):** ProaktivAI (runs inside IO)
- **Flagship product experience:** InqubatorAI (built on IO)

This repository is the initial home for platform architecture, roadmap, and implementation.
